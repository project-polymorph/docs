# 多元性别中文数字图书馆 - 文档库

多元性别中文数字图书馆创建于 **2024 年**，致力于收录、保存并展示与多元性别相关的所有中文资料。我们关注跨性别、非二元性别、间性人，以及其他打破传统二元性别认知的多元群体，包含数十万份相关资料。作为一个非营利、开放共享的数字图书馆，我们希望在日益严苛的审查环境和不断消逝的中文数字世界中，守护多元性别社群的宝贵记忆与经验。
